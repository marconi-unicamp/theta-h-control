# Supplementary Material

This folder contains additional figures and the compiled PDF of the supplementary document.

## 📁 Files

- `supplementary.pdf` – compiled PDF with extra analyses and figures.
- `fig_spectrum_comparison.png` – comparison of power spectra (Hann, Blackman, Kaiser).
- `fig_architecture.png` – Theta-H controller architecture diagram.
- `fig_simulink_diagram.png` – Simulink model overview.
- `fig_torus_3d.png` – 3D torus visualization.
- `fig_recurrence_plot.png` – recurrence plot (mosaic).
- `fig_poincare_phase.png` – Poincaré section colored by phase.

All figures were generated by scripts in `../code/generate_figures/`.
